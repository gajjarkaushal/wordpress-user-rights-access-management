WP-URAM is a simple plugin that helps you to easily restrict or set limited access on your admin menus, submenus, post types and custom meta boxes to logged-in users only.

It will work for both, Roles as well as Users.

Mainly this plugin will be very useful when your site is having multiple users and you wants to restrict these users with different different criterias.


== Description ==


This plugin provides an interface to restrict the admin menus, submenus post types and custom meta boxes; this plugin will work only for registered users.

There are 2 ways to limit access. 

Role wise - used to limit access of all users having same role.

User wise - used to limit access of a particular user.

User wise restriction can override the settings of role wise restriction.

Go to role or user tab and disable menu to restrict.


== Installation ==


Upload the WP-URAM plugin to your /wp-content/plugins/ directory.

Activate the plugin through the ‘Plugins’ menu in WordPress.

Done!
